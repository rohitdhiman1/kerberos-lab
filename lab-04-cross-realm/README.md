# Lab 04: Cross-Realm Trust

**Status:** Coming Soon  
**Difficulty:** Advanced  
**Prerequisites:** Lab 01

## Overview

Build a multi-realm Kerberos environment with cross-realm trust relationships.

## What You'll Learn

- Setting up multiple independent KDCs
- Configuring cross-realm principals
- Trust relationship establishment
- Multi-realm authentication flows
- Trust path verification

## Planned Components

- Two KDC containers (REALM-A and REALM-B)
- Clients for each realm
- Cross-realm service examples

## Note on Independence

This lab is **completely self-contained** with its own KDCs. It does not depend on any other lab's resources.
